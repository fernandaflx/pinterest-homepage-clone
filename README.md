## 📦 Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/fernandaflx/pinterest-homepage-clone.git
```

Entre no diretório do projeto

```bash
  cd pinterest-homepage-clone
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
